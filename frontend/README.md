# FashionFreak Frontend

React + Vite frontend for FashionFreak.

## Setup Instructions

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

The frontend will start on your local machine, generally at `http://localhost:5173`.

3. **Build for production:**
   ```bash
   npm run build
   ```
